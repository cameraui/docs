# Mobile Apps koppeln

Die [Mobile-Apps](/de/install/mobile) verbinden sich über camera.ui Cloud mit deinem Server, sodass du ihn von überall erreichst. Du koppelst die App einmal mit deinem Server.

## Koppeln

![Server-Kopplung mit QR-Code und Pair-Code](/img/remote/pairing.png)

1. Stelle sicher, dass [camera.ui Cloud](/de/remote/cloud) auf dem Server aktiviert ist.
2. Öffne auf dem Server **Einstellungen → Remote** und wähle **Server registrieren**. Ein **QR-Code**, ein **Pair-Code** und eine **Pair-URL** erscheinen.
3. Melde dich in der App an deinem camera.ui-Konto an und scanne dann den QR-Code oder gib den Pair-Code ein. Der Code ist begrenzt gültig.
4. Bestätige, und der Server erscheint in deinem Konto.

## Verbinden

Einmal gekoppelt, zeigt die App deine Server in der Cloud-Server-Liste. Wähle einen zum Verbinden. Hast du mehrere Server, kannst du in der App zwischen ihnen wechseln.
