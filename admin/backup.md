# Backup & restore

You can back up your camera.ui setup and restore it later, on any platform. It's in **Settings → Backup** (admins only).

## Creating a backup

Choose **Download backup archive** to save a single archive of your camera.ui configuration, database, and user accounts. Keep it somewhere safe.

## Restoring

Upload an archive and choose **Restore backup archive**. camera.ui restores your setup and then restarts to apply it. This replaces the current configuration, so be sure you want to roll back to the archive's state.

## Scheduled backups

Automatic, scheduled backups are planned but not available yet.[^sched]

[^sched]: The scheduled-backup option is shown in Settings but isn't functional yet.
